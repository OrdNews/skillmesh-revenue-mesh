## Full Description

SkillMesh Revenue Mesh is a self-funding agent organization built on X Layer for the OKX Build X Hackathon.

Instead of building a generic task bot or a simple marketplace, we built a system where:

- `Scout` reads market and treasury context
- `Trader` prepares a live route
- the organization spends treasury budget to hire specialist agents over `x402`
- `Treasury` records proof and mints a receipt on X Layer

The result is a product that does not just orchestrate agents. It lets an onchain organization hold treasury, deploy budget, buy intelligence, and leave verifiable proof.
